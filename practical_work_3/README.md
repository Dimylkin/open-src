# Практическая работа: Функции и область видимости (PHP)

## Описание

Учебный проект на PHP, реализующий набор функций для отработки базовых навыков работы с областями видимости, рекурсией, замыканиями и статической переменной. Код оформлен согласно стандартам PSR-12 и содержит подробные PHPDoc-комментарии.

## Содержимое

- Функция проверки простого числа (`isPrime`)
- Рекурсивное вычисление чисел Фибоначчи (`fibonacci`)
- Форматирование номера телефона (`formatPhone`)
- Анонимная фильтрация массива по чётности (`$filter`)
- Факториал с мемоизацией (`memoizedFactorial`)
- Создание пользователя с гибким вызовом аргументов (`createUser`)
- Генератор уникального счётчика, реализованный через замыкание (`makeCounter`)
- Примеры вывода результатов в HTML

## Установка и запуск

1. Склонируйте репозиторий в директорию веб-сервера:
    
    git clone https://github.com/Dimylkin/open-src.git   

2. Убедитесь, что используется PHP версии 8.0+.

3. Откройте файл в браузере по адресу:
    
    http://localhost/имя_директории/functions_deep.php
